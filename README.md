# Habit Hero

Habit Hero is a simple habit tracking application I built to help people stay consistent with their daily routines. The idea is pretty straightforward instead of just planning habits in your head, the app lets you create them, track them, and see how well you're sticking to them over time.

Most of us start new habits with good intentions, but after a few days we lose track. This project tries to solve that by giving a small dashboard where habits can be created, checked off, and monitored.



## What the application does

With Habit Hero, users can create habits they want to work on and track their progress day by day.

Some examples of habits could be:

- Exercising daily  
- Reading a book  
- Studying for a certain number of hours  
- Drinking enough water  

Once a habit is created, the user can mark it as completed for that day and optionally add notes about it. Over time the app records this data and shows progress patterns.



## Main Features

- Create new habits with a name, category, and frequency  
- Track daily or weekly completion of habits  
- View progress statistics like streaks and completion rate  
- Organize habits by categories such as health, productivity, or learning  



## Technologies Used

This project was built using a simple full-stack structure.

Frontend  
- ReactJS for the user interface and dashboard

Backend  
- Python with a web framework (Flask / FastAPI)

Database  
- SQLite for storing habits and progress data


## How to run the project

If you want to run the project locally, the basic steps are:

### 1. Clone the repository
git clone https://github.com/Himaya017/Habit-hero.git


### 2. Navigate to the project folder


cd Habit-hero


### 3. Run the backend server

Go to the backend folder and start the Python server.


cd backend
pip install -r requirements.txt
python main.py


### 4. Run the frontend

Open another terminal and run the React application.


cd frontend
npm install
npm start


The application should now open in your browser.
